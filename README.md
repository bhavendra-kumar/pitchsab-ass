# Pitchsap Landing Page Redesign

This project redesigns the Pitchsap landing page and implements a chat prototype for ideators.

## Features

- Modern responsive landing page
- Smooth animations
- Login & signup with JWT
- Protected dashboard
- Chat assistant for startup ideation

## Tech Stack

Frontend
- React
- TailwindCSS
- Framer Motion

Backend
- Node.js
- Express
- MongoDB
